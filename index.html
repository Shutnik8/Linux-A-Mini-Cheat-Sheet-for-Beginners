Блок 1 · Навигация Как перемещаться по системе
Работа с папками
Понимание текущего пути
Береги корень /
  • pwd где я
    Показать текущую директорию (полный путь).
  • ls список
    Показать файлы и папки. Часто используют ls -la для детального списка.
  • cd .. выше
    Перейти на уровень выше. cd / — в корень, cd ~ — в домашнюю.
  • cd ~/Downloads пример
    Переход в папку Downloads для текущего пользователя.
  • clear очистка
    Очистить окно терминала, чтобы не мешал старый вывод.
Блок 2 · Файлы и каталоги Создание, копирование, удаление
Опасно удалять без понимания
Всегда проверяй путь
  • mkdir my_folder папка
    Создать новую директорию my_folder в текущем каталоге.
  • touch file.txt файл
    Создать пустой файл file.txt или обновить время изменения.
  • cp source.txt dest.txt копия
    Скопировать файл в новый. Для папок добавь флаг -r (рекурсивно).
  • mv old.txt new.txt переименование
    Переместить или переименовать файл или папку: старое имя → новое.
  • rm file.txt удаление
    Удалить файл навсегда. Для папки используют rm -r folder, с флагом -f будь аккуратен.
Блок 3 · Система и помощь Что происходит и куда смотреть
Не бойся man
Сначала смотри справку
  • man ls справка
    Открыть руководство по команде ls.
  • history история
    Показать список ранее введенных команд в текущей оболочке.
  • cat file.txt просмотр
    Вывести содержимое файла в терминал. Для длинных файлов удобнее less file.txt.
  • ps aux процессы
    Показать список процессов в системе с подробной информацией.
  • sudo reboot перезапуск
    Перезагрузить систему с правами администратора.
