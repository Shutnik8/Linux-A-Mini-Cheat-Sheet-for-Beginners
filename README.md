# Linux-A-Mini-Cheat-Sheet-for-Beginners
[Linux: A Mini Cheat Sheet for Beginners](https://shutnik8.github.io/Linux-A-Mini-Cheat-Sheet-for-Beginners/)


Интерактивная шпаргалка по базовым командам Linux для новичков: яркий интерфейс, «живой» псевдо‑терминал, подсказки и модальные окна с описанием команд.

## Особенности

- Визуальная шпаргалка по основным командам: навигация, файлы/каталоги, система и справка.
- Клик по команде вставляет ее в строку псевдо‑терминала.  
- Двойной клик или ввод команды и нажатие Enter открывают модальное окно с подробным описанием и примерами.  
- Современный дизайн на чистых HTML, CSS и JavaScript, без зависимостей и сборщиков.

## Технологии

- **HTML5** — разметка и структура.  
- **CSS3** — адаптивный тёмный UI, градиенты, анимации.  
- **Vanilla JavaScript** — логика псевдо‑терминала, модальное окно, эффекты наведения.


## Для кого этот проект

- Новички, которые только знакомятся с Linux и терминалом.  
- Студенты и начинающие разработчики, которым нужен быстрый визуальный конспект.  
- Преподаватели, которые хотят показывать команды в более наглядном формате.
